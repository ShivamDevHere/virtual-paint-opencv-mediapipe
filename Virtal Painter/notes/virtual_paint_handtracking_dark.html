Virtual Paint using Hand Tracking

1️⃣ Hand Tracking Module

Purpose

Detect hand, track finger landmarks, and identify finger states (up/down).

Libraries Used

LibraryUse
OpenCVCamera input, drawing, display
MediaPipeHand detection and tracking
PythonProgram logic

Core Functions

FunctionUse
HandDetector()Initialize hand detection model
findHands(img)Detect hands and draw landmarks
FindPosition(img)Return 21 landmark positions
fingersUP()Detect which fingers are up

Hand Tracking Flow

Camera Frame
   ↓
Convert to RGB
   ↓
MediaPipe Hands.process()
   ↓
Detect landmarks (21 points)
   ↓
Convert to pixel coordinates
   ↓
Return landmark list

2️⃣ Virtual Paint Module

Purpose

Draw on the screen using finger gestures.

Libraries Used

LibraryUse
OpenCVDrawing, masking, display
NumPyCreate drawing canvas
OSLoad toolbar images
HandTrackingModuleGesture input

Important Variables

VariableUse

xp, ypPrevious finger position
drawcolorCurrent brush color

eraserThicknessEraser size

Canvas Creation

imgCanvas = np.zeros((720, 1280, 3), np.uint8)

Virtual Paint Flow

Read Camera Frame
   ↓
Flip Image
   ↓
Detect Hand
   ↓
Get Landmark Positions
   ↓
Check Fingers
   ↓
2 Fingers → Select Tool
1 Finger → Draw
   ↓
Update Canvas
   ↓
Merge Canvas with Frame
   ↓
Display Output
